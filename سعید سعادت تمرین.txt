سعید سعادت تمرین 
در زبان‌های برنامه‌نویسی مانند پایتون، print و return دو مفهوم متفاوت دارند و هر کدام کاربرد خاص خود را دارند. در زیر به توضیح هر یک و تفاوت‌های آن‌ها می‌پردازم:


▎۱. print

• تعریف: تابع print برای نمایش خروجی به کاربر در کنسول (یا ترمینال) استفاده می‌شود. این تابع مقداری را که به آن داده می‌شود، چاپ می‌کند و هیچ مقداری را برنمی‌گرداند.

• کاربرد: بیشتر برای دیباگ کردن یا نمایش اطلاعات به کاربر استفاده می‌شود.

مثال:
def greet(name):
    print("Hello, " + name)

greet("Alice")  # خروجی: Hello, Alice


در این مثال، تابع greet نام را چاپ می‌کند، اما هیچ مقداری را برنمی‌گرداند.

▎۲. return

• تعریف: عبارت return برای برگرداندن یک مقدار از یک تابع به محل فراخوانی آن استفاده می‌شود. وقتی یک تابع با return خاتمه می‌یابد، مقدار مشخص شده به عنوان خروجی تابع برمی‌گردد.

• کاربرد: برای استفاده از نتایج محاسبات در برنامه یا ذخیره‌سازی آن‌ها در متغیرها.

مثال:
def add(a, b):
    return a + b

result = add(3, 5)  # result برابر با 8 خواهد بود
print(result)       # خروجی: 8


در این مثال، تابع add دو عدد را با هم جمع می‌کند و نتیجه را با استفاده از return برمی‌گرداند. سپس این نتیجه در متغیر result ذخیره می‌شود و با استفاده از print چاپ می‌شود.

▎تفاوت‌های کلیدی

1. خروجی:

   • print: فقط خروجی را نمایش می‌دهد و هیچ مقداری را برنمی‌گرداند.

   • return: یک مقدار را برمی‌گرداند که می‌تواند در سایر قسمت‌های برنامه استفاده شود.

2. محل استفاده:

   • print: معمولاً برای نمایش اطلاعات به کاربر یا دیباگ کردن استفاده می‌شود.

   • return: برای بازگشت نتایج محاسبات از توابع به کار می‌رود.

3. توقف اجرای تابع:

   • print: اجرای تابع ادامه دارد حتی بعد از فراخوانی print.

   • return: بلافاصله اجرای تابع را متوقف می‌کند و کنترل را به محل فراخوانی بازمی‌گرداند.

با این توضیحات، امیدوارم تفاوت بین print و return به خوبی روشن شده باشد!
